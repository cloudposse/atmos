<CardGroup title="Describe Commands" className="cheatsheet">

    ```
    atmos describe affected
    ```

  </Card>

  <Card url="/cli/commands/describe/component">
    ```
    atmos describe component
    ```

  <Card url="/cli/commands/describe/config">
    ```
    atmos describe config
    ```

  </Card>

  <Card url="/cli/commands/describe/dependents">
    ```
    atmos describe dependents
    ```
    <p>Show a list of Atmos components in Atmos stacks that depend on the provided Atmos component</p>
  </Card>

  <Card url="/cli/commands/describe/stacks">
    ```
    atmos describe stacks
    ```
    <p>Show the fully deep-merged configuration for all Atmos stacks and the components in the stacks</p>

  <Card url="/cli/commands/describe/workflows">
    ```
    atmos describe workflows
    ```
    <p>Show the configured Atmos workflows</p>
  </Card>

<CardGroup title="Terraform Commands" className="cheatsheet">
  <Card url="/cli/commands/terraform/usage">
    ```
    atmos terraform
    ```
    <p>Execute `terraform` commands</p>
  </Card>

  <Card url="/cli/commands/terraform/clean">
    ```
    atmos terraform clean
    ```
    <p>Delete the `.terraform` folder, the folder that `TF_DATA_DIR` ENV var points to, `.terraform.lock.hcl` file, `varfile` and `planfile` for a component in a stack</p>
  </Card>

  <Card url="/cli/commands/terraform/deploy">
    ```
    atmos terraform deploy
    ```
    <p>Execute `terraform apply -auto-approve` on an Atmos component in an Atmos stack</p>
  </Card>

  <Card url="/cli/commands/terraform/generate-backend">
    ```
    atmos terraform generate backend
    ```
    <p>Generate a Terraform backend config file for an Atmos terraform component in an Atmos stack</p>
  </Card>

  <Card url="/cli/commands/terraform/generate-backends">
    ```
    atmos terraform generate backends
    ```
    <p>Generate the Terraform backend config files for all Atmos terraform components in all stacks</p>
  </Card>

  <Card url="/cli/commands/terraform/generate-varfile">
    ```
    atmos terraform generate varfile
    ```
    <p>Generate a varfile (`.tfvar` ) for an Atmos terraform component in an Atmos stack</p>
  </Card>

  <Card url="/cli/commands/terraform/generate-varfiles">
    ```
    atmos terraform generate varfiles
    ```
    <p>Generate the terraform varfiles (`.tfvar`) for all Atmos terraform components in all stacks</p>
  </Card>

  <Card url="/cli/commands/terraform/shell">
    ```
    atmos terraform shell
    ```
    <p>Start a new `SHELL` configured with the environment for an Atmos component in a stack to allow executing all native terraform commands inside the shell without using any atmos-specific arguments and flags</p>
  </Card>

  <Card url="/cli/commands/terraform/workspace">
    ```
    atmos terraform workspace
    ```
    <p>Calculate the Terraform workspace for an Atmos component (from the context variables and stack config), then run `terraform init -reconfigure`, then select the workspace by executing the `terraform workspace select` command</p>
  </Card>
</CardGroup>
