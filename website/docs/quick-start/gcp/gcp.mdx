# Google Cloud Platform (GCP) Tutorial

Take 30 minutes to learn the most important Atmos concepts for Google Cloud Platform (GCP).

:::tip

This Quick Start guide describes the steps to configure and provision the infrastructure
from this [Quick Start](https://github.com/cloudposse/atmos/tree/main/examples/quick-start-gcp) repository.

You can clone it and configure to your own needs. The repository should be a good start to get yourself familiar with Atmos.

:::

## Overview

The 10 steps to configure and provision the infrastructure are as follows:

1. [Install Atmos](/install)
2. [Configure Project Repository](/quick-start/gcp/configure-repository)
3. [Configure Atmos CLI](/quick-start/gcp/configure-cli)
4. [Vendor Component Dependencies](/quick-start/gcp/vendor-components)
5. [Create Atmos Stacks](/quick-start/gcp/create-atmos-stacks)
6. [Configure Validation](/quick-start/gcp/configure-validation)
7. [Automate Common Workflows](/quick-start/gcp/create-workflows)
8. [Add Custom Commands](/quick-start/gcp/add-custom-commands)
9. [Configure Terraform Backend](/quick-start/gcp/configure-terraform-backend)
10. [Deploy Everything](/quick-start/gcp/provision)

We'll then conclude with some [final notes](/quick-start/gcp/final-notes) and [next steps](/quick-start/gcp/next-steps)
