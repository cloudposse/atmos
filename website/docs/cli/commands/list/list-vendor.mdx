# atmos list vendor

The `atmos list vendor` command displays information about vendored components in your Atmos project.

## Usage

```shell
atmos list vendor [flags]
```

## Description

The `atmos list vendor` command helps you manage and inspect vendored components in your Atmos project. It provides information about:

- Vendored component locations
- Source repositories
- Version information
- Component status

This command is particularly useful when working with vendored components to:
- Track which components are vendored in your project
- Verify component versions and sources
- Manage component dependencies
- Ensure consistency across your infrastructure

## Flags

| Flag | Description |
|------|-------------|
| `-h, --help` | Help for the vendor command |
| `-o, --format string` | Output format: table, json, csv, tsv (default "table") |
| `--delimiter string` | Delimiter for csv/tsv output (default: comma for csv, tab for tsv) |

## Examples

List all vendored components in table format (default):
```shell
atmos list vendor
```

List vendored components in JSON format:
```shell
atmos list vendor --format json
```

List vendored components in CSV format:
```shell
atmos list vendor --format csv
```

List vendored components in TSV format:
```shell
atmos list vendor --format tsv
```

List vendored components in CSV format with custom delimiter:
```shell
atmos list vendor --format csv --delimiter ";"
```
