:::note Purpose
Use this command to list Atmos stacks. For detailed stack information and rich output formatting, use `atmos describe stacks`.
:::

<Screengrab title="atmos list stacks --help" slug="atmos-list-stacks--help" />

## Usage

Execute the `list stacks` command like this:

```shell
# List all stacks
atmos list stacks

# List stacks for a specific component
atmos list stacks -c <component>
```

:::tip
Run `atmos list stacks --help` to see all the available options
:::

## Examples

```shell
# List all stacks
atmos list stacks

# List stacks for a specific component
atmos list stacks -c vpc
```

## Flags

| Flag             | Description                                                                                                                                                                                                                               | Alias | Required |
|------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------|----------|
| `--component`    | Filter stacks by component                                                                                                                                                                                                              | `-c`  | no       |

:::tip
For detailed stack information, JSON output, and advanced formatting options, use the `describe` command:
```shell
# Show detailed stack information
atmos describe stacks

# Show detailed information for a specific stack
atmos describe stack <stack-name>
```
:::
