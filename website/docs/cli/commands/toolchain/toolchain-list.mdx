<Intro>
Display all CLI tools registered in your toolchain configuration. This gives you a quick overview of which tool versions are available for use in your components, commands, and workflows.
</Intro>

<Screengrab title="atmos toolchain list --help" slug="atmos-toolchain-list--help" />

## Usage

Execute the `atmos toolchain list` command like this:

```shell
atmos toolchain list
```

## Examples

```shell
atmos toolchain list
```
