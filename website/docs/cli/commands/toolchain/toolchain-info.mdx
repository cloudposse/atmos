<Intro>
Display detailed information about installed toolchain tools, including version, installation path, and metadata. This helps you verify that the correct tools are available for your components and workflows.
</Intro>

<Screengrab title="atmos toolchain info --help" slug="atmos-toolchain-info--help" />

## Usage

Execute the `atmos toolchain info` command like this:

```shell
atmos toolchain info <tool>
```

## Examples

```shell
atmos toolchain info terraform
```

## Arguments

<dl>
    <dt>`package` <em>(required)</em></dt>
    <dd>
        Package to install from the aqua repository
    </dd>
</dl>

## Flags

<dl>
    <dt>`--output`  <em>(optional)</em></dt>
    <dd>
        Output format table or json
    </dd>

</dl>
