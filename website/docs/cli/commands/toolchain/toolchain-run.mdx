## Usage

Execute the `toolchain run` command like this:

```shell
atmos toolchain run <tool> <arguments>
```

## Examples

```shell
atmos toolchain run terraform --version         # Uses latest version
atmos toolchain run terraform@1.9.8 --version   # Uses specific version
atmos toolchain run opentofu@1.10.1 init
atmos toolchain run terraform@1.5.7 plan -var-file=prod.tfvars```

## Arguments

<dl>
    <dt>`package` <em>(required)</em></dt>
    <dd>
        Package to install from the aqua repository
    </dd>
    <dt>`version` <em>(required)</em></dt>
    <dd>
        The version to add and update
    </dd>
</dl>

## Flags

<dl>
    <dt>`--file`  <em>(optional)</em></dt>
    <dd>
        Path to tool-versions file (defaults to global --tool-versions-file)
    </dd>

</dl>
