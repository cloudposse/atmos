<Intro>
Use the `atmos lint` commands to analyze Atmos stack configurations for anti-patterns, optimization opportunities, and structural issues. Linting is distinct from `atmos validate stacks`: validation checks correctness, while linting checks quality, DRY-ness, and best practices.
</Intro>

## Usage

```shell
atmos lint [subcommand] [flags]
```
