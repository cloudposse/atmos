## Usage

```shell
atmos cast render <input.cast> [--svg <file>] [--gif <file>] [--mp4 <file>]
```

At least one output flag is required.

## Arguments

<dl>
  <dt>`input.cast`</dt>
  <dd>**Required.** Path to the asciicast recording to render.</dd>
</dl>

## Flags

<dl>
  <dt>`--svg`</dt>
  <dd>Write animated SVG output to this path.</dd>

  <dt>`--gif`</dt>
  <dd>Write animated GIF output to this path.</dd>

  <dt>`--mp4`</dt>
  <dd>Write MP4 output to this path.</dd>
</dl>

## Examples

```shell
atmos cast render ./artifacts/demo.cast --gif ./artifacts/demo.gif
atmos cast render ./artifacts/demo.cast --gif ./artifacts/demo.gif --mp4 ./artifacts/demo.mp4
```
