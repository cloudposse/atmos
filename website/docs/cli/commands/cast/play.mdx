## Usage

```shell
atmos cast play <input.cast>
```

## Arguments

<dl>
  <dt>`input.cast`</dt>
  <dd>**Required.** Path to the asciicast recording to replay.</dd>
</dl>

## Examples

```shell
atmos cast play ./artifacts/demo.cast
```
