## Usage

```bash
atmos support
```

## Description

The `support` command displays all available support channels for getting help with Atmos:

### Community Resources

- **Slack Community**: Join the `#atmos` channel at [cloudposse.com/slack](https://cloudposse.com/slack)
- **GitHub Discussions**: Post questions or join conversations with the community
- **Issue Tracker**: Report bugs at [github.com/cloudposse/atmos/issues](https://github.com/cloudposse/atmos/issues)

### Office Hours

Cloud Posse hosts **office hours every Wednesday at 11:30 AM PST** for live Q&A sessions.

[cloudposse.com/office-hours](https://cloudposse.com/office-hours)

### Paid Support

Priority support is available to GitHub Sponsors (Enterprise Tier):

[github.com/sponsors/cloudposse](https://github.com/sponsors/cloudposse)

Sponsorship includes **30-minute workshops twice a week** to:

- Answer your questions
- Assist with debugging issues
- Discuss architectural decisions
