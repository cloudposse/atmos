:::note Purpose
Interactively configure static AWS user credentials (AWS Access Key ID, Secret Access Key, and optional MFA ARN) and store them securely in your OS keychain. Only identities with `kind: aws/user` are supported.
:::

<Screengrab title="atmos auth user configure --help" slug="atmos-auth-user-configure--help" />

## Usage

```shell
atmos auth user configure
```

## Description

- Prompts for `AWS Access Key ID`, `AWS Secret Access Key`, and optional `AWS User MFA ARN`.
- Stores credentials securely in the OS keychain using an alias matching the selected `aws/user` identity name.
- Only identities with `kind: aws/user` are selectable.

## Examples

```shell
# Configure credentials for an aws/user identity
atmos auth user configure
```
