<Intro>
Use this command to run `rain build` through Atmos. Atmos can resolve component context, authentication, and toolchain dependencies before passing arguments to Rain.
</Intro>

## Usage

```shell
atmos rain build [component] --stack <stack> [atmos-flags] -- [rain-options]
atmos rain build -- [rain-options]
```

## Arguments

<dl>
  <dt>`component` <em>(optional)</em></dt>
  <dd>Rain component used to resolve Atmos stack context, auth, dependencies, and working directory before running `rain build`.</dd>
</dl>

## Examples

<Terminal>
```shell
# Run with component context
atmos rain build api -s dev -- --help

# Run Rain directly after loading Atmos CLI configuration
atmos rain build -- --help
```
</Terminal>
