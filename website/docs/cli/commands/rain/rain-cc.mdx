<Intro>
Use this command to run `rain cc` through Atmos. Atmos can prepare stack configuration, authentication, and toolchain dependencies before invoking Rain.
</Intro>

## Usage

```shell
atmos rain cc [component] --stack <stack> [atmos-flags] -- [rain-options]
atmos rain cc -- [rain-options]
```

## Arguments

<dl>
  <dt>`component` <em>(optional)</em></dt>
  <dd>Rain component used to resolve Atmos stack context, auth, dependencies, and working directory before running `rain cc`.</dd>
</dl>

## Examples

<Terminal>
```shell
atmos rain cc api -s dev -- --help
atmos rain cc -- --help
```
</Terminal>
