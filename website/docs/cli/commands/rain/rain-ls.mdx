<Intro>
Use this command to run `rain ls` through Atmos. Atmos can prepare stack configuration, authentication, and toolchain dependencies before invoking Rain.
</Intro>

<Screengrab title="atmos rain ls --help" slug="atmos-rain-ls--help" />

## Usage

```shell
atmos rain ls [component] --stack <stack> [atmos-flags] -- [rain-options]
atmos rain ls -- [rain-options]
```

## Arguments

<dl>
  <dt>`component` <em>(optional)</em></dt>
  <dd>Rain component used to resolve Atmos stack context, auth, dependencies, and working directory before running `rain ls`.</dd>
</dl>

## Examples

<Terminal>
```shell
atmos rain ls api -s dev
atmos rain ls -- --all
```
</Terminal>
