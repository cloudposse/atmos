<Intro>
Use this command to run `rain merge` through Atmos. Atmos can prepare stack configuration, authentication, and toolchain dependencies before invoking Rain.
</Intro>

## Usage

```shell
atmos rain merge [component] --stack <stack> [atmos-flags] -- [rain-options]
atmos rain merge -- [rain-options]
```

## Arguments

<dl>
  <dt>`component` <em>(optional)</em></dt>
  <dd>Rain component used to resolve Atmos stack context, auth, dependencies, and working directory before running `rain merge`.</dd>
</dl>

## Examples

<Terminal>
```shell
atmos rain merge api -s dev -- --help
atmos rain merge -- --help
```
</Terminal>
