<Intro>
Use this command to remove the CloudFormation stack named by an Atmos Rain component.
</Intro>

## Usage

```shell
atmos rain rm <component> --stack <stack> [atmos-flags] -- [rain-options]
atmos rain rm --all [-s <stack>] [atmos-flags]
atmos rain rm --affected [git-flags] [atmos-flags]
```

## Arguments

<dl>
  <dt>`component` <em>(required for single-component runs)</em></dt>
  <dd>Atmos Rain component. The component must define `name`, which Atmos passes to Rain as the CloudFormation stack name.</dd>
</dl>

## Examples

<Terminal>
```shell
# Remove one stack
atmos rain rm api -s dev

# Remove all Rain stacks in reverse dependency order
atmos rain rm --all -s dev

# Authenticate before removal
atmos rain rm api -s dev --identity prod-admin
```
</Terminal>

## Stack Configuration

```yaml
components:
  rain:
    api:
      name: app-api-dev
```
