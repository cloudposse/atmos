<Intro>
Use this command to run `rain watch` for the CloudFormation stack named by a Rain component.
</Intro>

## Usage

```shell
atmos rain watch <component> --stack <stack> [atmos-flags] -- [rain-options]
```

## Arguments

<dl>
  <dt>`component` <em>(required)</em></dt>
  <dd>Atmos Rain component. The component must define `name`, which Atmos passes to Rain as the CloudFormation stack name.</dd>
</dl>

## Examples

<Terminal>
```shell
atmos rain watch api -s dev
atmos rain watch api -s dev --identity prod-admin
```
</Terminal>

## Stack Configuration

```yaml
components:
  rain:
    api:
      name: app-api-dev
```
