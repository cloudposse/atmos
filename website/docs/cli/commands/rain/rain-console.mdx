<Intro>
Use this command to run `rain console` for the CloudFormation stack named by a Rain component.
</Intro>

<Screengrab title="atmos rain console --help" slug="atmos-rain-console--help" />

## Usage

```shell
atmos rain console <component> --stack <stack> [atmos-flags] -- [rain-options]
```

## Arguments

<dl>
  <dt>`component` <em>(required)</em></dt>
  <dd>Atmos Rain component. The component must define `name`, which Atmos passes to Rain as the CloudFormation stack name.</dd>
</dl>

## Flags

<dl>
  <dt>`--stack`, `-s` <em>(required)</em></dt>
  <dd>Atmos stack.</dd>

  <dt>`--identity`, `-i` <em>(optional)</em></dt>
  <dd>Authenticate with an Atmos identity before stack processing and before invoking Rain.</dd>
</dl>

## Examples

<Terminal>
```shell
atmos rain console api -s dev
atmos rain console api -s dev --identity prod-admin
```
</Terminal>

## Stack Configuration

```yaml
components:
  rain:
    api:
      name: app-api-dev
```
