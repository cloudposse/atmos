<Intro>
Use this command to run `rain stackset` through Atmos. Atmos can prepare stack configuration, authentication, and toolchain dependencies before invoking Rain.
</Intro>

## Usage

```shell
atmos rain stackset [component] --stack <stack> [atmos-flags] -- [rain-options]
atmos rain stackset -- [rain-options]
```

## Arguments

<dl>
  <dt>`component` <em>(optional)</em></dt>
  <dd>Rain component used to resolve Atmos stack context, auth, dependencies, and working directory before running `rain stackset`.</dd>
</dl>

## Examples

<Terminal>
```shell
atmos rain stackset api -s dev -- --help
atmos rain stackset -- --help
```
</Terminal>
