<Intro>
Use this command to run `rain module` through Atmos. Atmos can prepare stack configuration, authentication, and toolchain dependencies before invoking Rain.
</Intro>

<Screengrab title="atmos rain module --help" slug="atmos-rain-module--help" />

## Usage

```shell
atmos rain module [component] --stack <stack> [atmos-flags] -- [rain-options]
atmos rain module -- [rain-options]
```

## Arguments

<dl>
  <dt>`component` <em>(optional)</em></dt>
  <dd>Rain component used to resolve Atmos stack context, auth, dependencies, and working directory before running `rain module`.</dd>
</dl>

## Examples

<Terminal>
```shell
atmos rain module api -s dev -- --help
atmos rain module -- --help
```
</Terminal>
