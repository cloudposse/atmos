:::note Purpose
Use these commands to provision infrastructure resources that Terraform depends on, such as S3 buckets for state storage.
:::

## Usage

```shell
atmos terraform provision <subcommand> [options]
```

## Available Subcommands

<dl>
  <dt>[`backend`](/cli/commands/terraform/terraform-provision-backend)</dt>
  <dd>Manage Terraform state backends (create, list, describe, update, delete)</dd>
</dl>

## Examples

### Provision Backend Infrastructure

```shell
atmos terraform provision backend create vpc --stack dev
```

### List All Backends

```shell
atmos terraform provision backend list --stack dev
```
