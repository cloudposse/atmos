Remove a dependency entry from a version track in `atmos.yaml`, preserving comments and formatting. The alias is `rm`.

## Usage

```shell
atmos version track remove NAME [flags]
atmos version track rm NAME [flags]
```

## Arguments

<dl>
  <dt>`NAME` <em>required</em></dt>
  <dd>Entry name to remove.</dd>
</dl>

## Examples

```shell
# Remove nginx from the default track
atmos version track remove nginx

# Remove nginx from prod
atmos version track rm nginx --track=prod
```
