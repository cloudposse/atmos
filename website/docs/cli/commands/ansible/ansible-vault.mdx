:::note Purpose
Use this command to create, edit, encrypt, decrypt, and view encrypted files using Ansible Vault.
This allows you to keep sensitive data such as passwords or keys in encrypted files.
:::

<Screengrab title="atmos ansible vault --help" slug="atmos-ansible-vault--help" />

## Usage

```shell
atmos ansible vault <component> --stack <stack> [action] [file] [options]
```

## Examples

```shell
atmos ansible vault webapp --stack dev encrypt secrets.yml
atmos ansible vault webapp --stack dev decrypt secrets.yml
atmos ansible vault webapp -s prod create new-secrets.yml
atmos ansible vault webapp -s prod edit secrets.yml
```

## Arguments

<dl>
  <dt>`component`</dt>
  <dd>Atmos component name</dd>
  
  <dt>`action`</dt>
  <dd>Vault action (encrypt, decrypt, create, edit, view, etc.)</dd>
  
  <dt>`file`</dt>
  <dd>File to perform the vault action on</dd>
</dl>

## Flags

<dl>
  <dt>`--stack` / `-s`</dt>
  <dd>Atmos stack (required)</dd>
</dl>

:::tip
For more details on the `ansible-vault` command and options, refer to [Ansible vault command reference](https://docs.ansible.com/ansible/latest/cli/ansible-vault.html).
:::
