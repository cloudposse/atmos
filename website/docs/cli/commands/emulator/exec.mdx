<Intro>
Use this command to run a command inside the emulator's container. Pass the command and its arguments after `--`. If no command is supplied, a shell is opened. The container is discovered by label.
</Intro>

<Experimental />

<CastPlayer title="atmos emulator exec --help" src="/casts/screengrabs/atmos-emulator-exec--help.cast" static chrome />

## Usage

```shell
atmos emulator exec <component> --stack <stack> -- <command> [args...]
```

## Examples

```shell
# List S3 buckets in the local aws emulator
atmos emulator exec aws --stack=plat-ue2-dev -- aws s3 ls

# Open a shell in the emulator container (no command after --)
atmos emulator exec aws --stack=plat-ue2-dev

# Using the emu alias
atmos emu exec aws --stack=plat-ue2-dev -- env
```

## Arguments

<dl>
  <dt>`component`</dt>
  <dd>**Required.** The emulator component to run the command in (for example, `aws`).</dd>

  <dt>`command` <em>(after `--`)</em></dt>
  <dd>The command and arguments to run inside the container. Everything after `--` is passed through to the container verbatim. If omitted, a shell is opened.</dd>
</dl>

## Flags

<dl>
  <dt>`-s, --stack`</dt>
  <dd>**Required.** The stack the emulator component belongs to (for example, `plat-ue2-dev`).</dd>
</dl>
