<Intro>
The `aws` section of the `atmos.yaml` configures AWS-specific features that integrate directly
with AWS services. Each subsection controls a different AWS capability.
</Intro>

<Experimental />

## Quick Start

<File title="atmos.yaml">
```yaml
aws:
  security:
    enabled: true
```
</File>

## Configuration Reference

### Top-Level Structure

<File title="atmos.yaml">
```yaml
aws:
  # Security scanning and compliance reporting
  security:
    enabled: true
    # ... see Security documentation for full options
```
</File>
