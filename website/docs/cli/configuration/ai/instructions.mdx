<Intro>
The `settings.ai.instructions` section configures the `ATMOS.md` file that provides persistent, project-specific
instructions to the AI assistant.
</Intro>

<Experimental />

## Configuration

<File title="atmos.yaml">
```yaml
settings:
  ai:
    instructions:
      enabled: true
      file: "ATMOS.md"
```
</File>

## Instruction Settings

<dl>
  <dt>`enabled`</dt>
  <dd>Enable or disable project instructions (default: `true`)</dd>

  <dt>`file`</dt>
  <dd>Path to the instructions file relative to the project root (default: `ATMOS.md`)</dd>
</dl>

For ATMOS.md file structure and best practices, see [Project Instructions](/ai/instructions).
