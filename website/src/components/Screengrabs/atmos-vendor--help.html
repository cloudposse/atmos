Usage: atmos vendor [sub-command] [flags] Available Commands: pull Pull the latest vendor configurations or dependencies Flags: -h, --help help for vendor Global Flags: --base-path string Base path for Atmos project --config stringSlice Paths to configuration files (comma-separated or repeated flag) (default [] ) --config-path stringSlice Paths to configuration directories (comma-separated or repeated flag) (default [] ) --logs-file string The file to write Atmos logs to.
