Usage: atmos packer [sub-command] [flags] Aliases: packer, pk Available Commands: build Build a machine image from a Packer configuration. init Initialize Packer according to an HCL template configuration. inspect Inspect a Packer configuration. output Get an output from a Packer manifest. validate Validate a Packer template. version Show Packer version.
