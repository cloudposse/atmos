Usage: atmos helmfile generate [sub-command] [flags] Available Commands: varfile Generate a values file for a Helmfile component Flags: -h, --help help for generate (default "false") Global Flags: -- Use double dashes to separate Atmos-specific options from native arguments and flags for the command. --logs-file string The file to write Atmos logs to.
