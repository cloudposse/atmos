This command allows you to ask questions directly from the command line without entering an interactive chat session. The AI has access to your Atmos configuration and can provide context-aware responses. Examples: atmos ai ask "What components are available?" atmos ai ask "How do I validate my stack configuration?" atmos ai ask "Explain the difference between components and stacks" atmos ai ask "Describe the vpc component in the dev stack" USAGE $ atmos ai ask [question] [flags] EXAMPLES - Ask the AI assistant a question directly from the command line $ atmos ai ask "What components are available?" $ atmos ai ask "How do I validate my stack configuration?" $ atmos ai ask "Explain the difference between components and stacks" $ atmos ai ask "Describe the vpc component in the dev stack" $ atmos ai ask "What are the best practices for organizing stacks?"
