Usage: atmos describe dependents [flags] Aliases: dependents, dependants Flags: --file string Write the result to the file: atmos describe dependents <component> -s <stack> --file dependents.yaml -f, --format string The output format: atmos describe dependents <component> -s <stack> --format=json|yaml ('json' is default) (default "json") -h, --help help for dependents (default "false") -s, --stack string atmos describe dependents <component> -s <stack> Global Flags: --logs-file string The file to write Atmos logs to. Supported log levels are Trace, Debug, Info, Warning, Off.
