## Atmos Command Targets
```text
Common repository commands:

  atmos build deps                    Download Go module dependencies
  atmos build                  Build to ./build/atmos
  atmos lint changed                  Lint changed files
  atmos test short                    Run quick tests
  atmos test acc                      Run acceptance tests
  atmos screengrabs build about       Generate matching screengrabs
  atmos screengrabs build --all       Generate screengrabs
  atmos screengrabs all               Generate and install screengrabs
```
