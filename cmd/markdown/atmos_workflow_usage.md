– Use interactive UI

```shell
 $ atmos workflow
```

– Execute a workflow

```shell
 $ atmos workflow <workflow-name> --file <file>
```

– Execute with stack override

```shell
 $ atmos workflow <workflow-name> --file <file> --stack <stack>
```

– Resume from specific step

```shell
 $ atmos workflow <workflow-name> --file <file> --from-step <step>
```
