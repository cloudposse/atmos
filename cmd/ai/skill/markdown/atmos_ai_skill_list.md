List Atmos skills — both the official skills bundled with Atmos and any
community skills installed on this system.

A filled dot (●) marks an installed skill; a hollow dot (○) marks one that is
available to install. The Source column shows where the skill comes from:
`built-in` for official skills embedded in Atmos, or a repository source for
community installs. Install built-in skills by Name; install community skills
from their repository source. Installed skills show their version. Use
`--installed` to show only installed skills, and `--detailed` for full per-skill
information.

Installed skills are stored in ~/.atmos/skills/ and tracked in registry.json.
The official catalog is embedded in the Atmos binary, so listing works offline.
